# ユースケースガイド

## 目的

finger-tracker の操作手順をユーザー視点で記録する。

### 対象読者

- システムの操作者（研究者）

## ディレクトリ構成

```
usecases/
├── GUIDE.md           # 本ガイド
├── TEMPLATE.md        # ユースケーステンプレート
└── {usecase}.md       # 個別ユースケース
```

## 種別

| 種別 | 説明 | 例 |
|------|------|---|
| データ収集 | 学習データの取得手順 | RealSenseで指サック画像を撮影 |
| モデル学習 | 学習・評価の手順 | YOLOv8-nanoのfine-tuning |
| 計測 | リアルタイム計測の操作手順 | 指間距離計測の実行 |

## セクション構成

| セクション | 必須/推奨 |
|-----------|----------|
| 概要 | 必須 |
| 前提条件 | 必須 |
| 操作手順 | 必須 |
| 期待結果 | 必須 |
| トラブルシューティング | 推奨 |

## 記法規約

### 操作フロー

ステップ番号 + 期待結果の形式で記述:

```markdown
1. RealSense D435i をUSB接続する
   - **期待結果**: `rs-enumerate-devices` でデバイスが表示される
2. データ収集スクリプトを実行する
   ```bash
   python -m finger_tracker.capture
   ```
   - **期待結果**: カメラ映像が表示され、画像が `data/images/` に保存される
```

## テンプレートの使用

`TEMPLATE.md` をコピーし、プレースホルダを埋める。ファイル名: `{kebab-case-usecase}.md`

## 品質チェックリスト

- [ ] 操作手順が順序通りに記述されている
- [ ] 各ステップに期待結果がある
- [ ] 前提条件（ハードウェア・ソフトウェア）が明記されている
- [ ] コマンドがコピーペースト可能
